＜概要＞
　リバーシを対局するプログラムを作成するための解説です。
　ソースコードも付属しています。

＜プログラム動作環境＞
　OS: Windows XP

＜インストール／アンインストール方法＞
　・インストール方法
　　zipファイルを解凍してください
　・アンインストール方法
　　zipファイル解凍によって作成されたディレクトリ、ファイルを全て削除してください

＜使用条件＞
　本ソフトウェアはフリーソフトウェアです。
　私的な使用、改変は自由に行えます。

＜禁止事項＞
　以下の行為を禁じます
　・ソフトウェアに含まれるファイル全部または一部を再配布すること。
　・ソフトウェアに含まれるソースコードを改変後、そのまままたはコンパイル、リンクした形状で配布すること。
　・ソフトウェアに含まれるテキストファイルまたはHTMLファイルを改変後配布すること。

　※ソフトウェアに含まれる定石データおよび評価パラメータデータにつきましては、改変後であれば配布することを許可します。

＜サンプルプログラム使用方法＞
・コンソール版
(1)reversi.exeを実行してください。
(2)モードを入力してください。
　"q"を入力すると終了します。
　対局したいときには対戦モードを選択してください。
　学習をさせたい場合には、学習モードを選択して学習した後、MPC計算モードを選択してMPC計算を行なってください。
　学習時の対局数は20000〜100000程度が良いと思います。
(3.1 対戦モード)
　自分の色、コンピュータのレベルを入力してください。
　対局が開始されるので以下のコマンドを入力してください。
　・座標（例：F5）
　　着手を行ないます。
　・"u"または"undo"
　　１手戻します
　・"p"または"pass"
　　パスします。ルール上パスできる場合にのみ有効です。
　・"q"または"quit"
　　対局を中断します。
　　中断した局面から再開することはできません。
(3.2 学習モード)
　学習時の対局数を入力してください。
　入力後コンピュータ同士で対局を行い、「data」フォルダにある「eval.dat」の更新を行ないます。
　100局毎に進捗が表示され、「eval.dat」が更新されます。
　参考までに、Athlon XP 2600+では100局学習するのに40秒程度かかりました。
　サンプル版の探索手数は6/14/14（中盤/完全/必勝）で、変更はできません。
　0から学習させたい場合には「eval.dat」を削除してください。
　「eval.dat」を初期時に戻したい場合には、「data/org」フォルダにある「eval.dat」を
　「data」フォルダにコピーしてください。
(3.3 定石登録モード)
　定石登録を行ないます。
　新たに定石を登録したい場合にはdataフォルダにある「open_trans.txt」を編集してください。
　フォーマットは以下の通りです。

（棋譜）
（黒番から見た棋譜の評価値）
（上記２行の繰り返し）

例：
F5D6C3D3C4F4C5B3C2E6C6B4
0
F5D6C3D3C4F4C5B3C2B4E3E6
0

(3.4 MPC計算モード)
　中盤で枝刈りを行なうためのパラメータ計算を行ないます。
　学習を行なった後には必ずMPC計算を行なってください。

・GUI版
　(1)reversi_gui.exeを実行してください。
　(2)「対局」メニューから自分の色、コンピュータのレベルを選択してください。
　(3)「対局」-「対局開始」で対局を開始します。
　(4)対局中は、着手したいマスをクリックしてください。
　(5)「対局」-「中断」で対局を中断します。
　　中断した局面から再開することはできません。

　コンピュータの探索手数は以下の通りです。（中盤読み手数/完全読み手数/必勝読み手数）
　ご参考までに
　レベル１：2/8/10
　レベル２：4/10/12
　レベル３：6/12/14
　レベル４：8/14/16
　レベル５：10/16/18
　レベル６：12/16/18
　レベル７：14/18/20

＜コンパイル方法＞
　ソースファイルをコンパイルするためにはCコンパイラが必要です。
　以下の環境でのプロジェクトファイルまたはMakefileを用意していますのでご利用ください。
・Visual C++ 6.0
　reversi.dswをVisual C++で開いてください
・gcc
　makefile.gccがgcc用Makefileです。
　make -f makefile.gcc allでコンパイルできます。
　make -f makefile.gcc cleanでコンパイル時に作成した*.oを削除します。
・Borland C++ Compiler 5.5
　makefile.bccがBorland C++ Compiler 5.5用Makefileです。
　make -f makefile.bcc allでコンパイルできます。
　make -f makefile.bcc cleanでコンパイル時に作成した*.objと*.tdsを削除します。
