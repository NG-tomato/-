はじめに

本解説の目的は「シンプルな実装である程度強いリバーシプログラムを作る」ことです。
以下のような方を対象にしています。

  • 手軽に強いリバーシプログラムを作りたい方
  • 思考アルゴリズムの基礎を知りたい方
以下のような方に対しては、手助けにはなるかもしれませんが不十分です。
他の資料も併せて参照されることをお勧めします。
  • 世界最強のリバーシプログラムを作りたい方
  • C言語の書き方について勉強したい方

記述言語はC言語を採用しています。
筆者がよく使うというのが最大の理由ですが、多くのプラットフォームでコンパイル、実行ができるというのも理由の１つです。
また他の言語に比べると比較的多くの人が理解できると思います。

ソースコードは以下の開発環境でコンパイルできることを確認しています。

  • Visual C++ 6.0
  • gcc 3.4.2
  • Borland C++ Compiler 5.5
特に環境依存がないように作成したつもりですが、特定の環境でコンパイルエラーが発生する等の現象がございましたら、ご連絡ください。
